# halo-app
[Halo](https://halo.run/) admin app client
一个[Halo](https://halo.run/)的移动客户端

### 开发框架
- [Uni-App](https://uniapp.dcloud.io/)

### HaloApi
- [HaloApi](https://api.halo.run/admin-api.html)


### 进度
目前还是刚刚开始，也顺便练练手
已经完成以下功能：

- 登录站点
- 显示个人信息
- 显示博客统计信息
- 显示最近文章
